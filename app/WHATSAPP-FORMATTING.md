# WhatsApp Message Formatting Guide

This guide explains how to format your messages to look great on WhatsApp.

## Text Formatting

WhatsApp supports several text formatting options:

### Bold Text
Wrap text with asterisks (*):
```
*This is bold text*
```
Result: **This is bold text**

### Italic Text
Wrap text with underscores (_):
```
_This is italic text_
```
Result: *This is italic text*

### Strikethrough Text
Wrap text with tildes (~):
```
~This is strikethrough~
```
Result: ~~This is strikethrough~~

### Monospace/Code Text
Wrap text with three backticks (```):
```
```This is monospace text```
```
Result: `This is monospace text`

## Combining Formatting

You can combine different formatting styles:

```
*_Bold and Italic_*
*~Bold Strikethrough~*
```

## Best Practices

### Do's ✅
- Keep messages concise (under 1000 characters)
- Use formatting sparingly for emphasis
- Add emojis to convey emotion
- Include clear call-to-action
- Use line breaks for readability
- Personalize with {{name}}
- Test your message before sending

### Don'ts ❌
- Don't use ALL CAPS (seems like shouting)
- Don't overuse formatting
- Don't spam emojis
- Don't send very long messages (max 4096 chars)
- Don't use too many links
- Don't forget to proofread

## Character Limits

- **WhatsApp limit:** 4,096 characters
- **Recommended:** 500-1000 characters
- **Optimal:** 200-500 characters

WhaSender shows character count as you type.

Before sending to all contacts:

1. **Preview**: Use WhaSender's preview feature
2. **Send Test**: Send to your own number first
3. **Check Formatting**: Verify all formatting renders correctly
4. **Check Links**: Ensure all links work
5. **Check Variables**: Confirm {{name}} replaces properly
6. **Read Aloud**: Does it sound natural?
7. **Check Length**: Is it too long?

## Common Mistakes to Avoid

### 1. Formatting Errors
❌ Wrong: `*Bold text` (missing closing asterisk)
✅ Correct: `*Bold text*`

### 2. Variable Syntax
❌ Wrong: `{name}` (single braces)
✅ Correct: `{{name}}` (double braces)

### 3. Overformatting
❌ Wrong:
```
*_~```LOOK AT THIS```~_*
```
✅ Better:
```
*LOOK AT THIS*
```

### 4. Too Many Emojis
❌ Wrong: `🎉🎊🎁🎈🎀💝✨🌟⭐ Special offer 🎉🎊🎁🎈`
✅ Better: `🎁 Special offer for you!`

## Character Count Tips

To stay within limits:

1. **Use abbreviations** (e.g., "Dr." instead of "Doctor")
2. **Remove filler words** ("just", "really", etc.)
3. **Use bullet points** instead of sentences
4. **Shorten URLs** (use bit.ly or similar)
5. **Split into multiple messages** if needed (not recommended)
